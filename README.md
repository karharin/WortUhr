# MillerWortUhr
Uhrenprojekt der Millers

## Aktuelle Version
Die aktuelle Version ist Uhrencode V3-2 WebBased. 
Diese Version ermöglicht es die Uhr über eine eigene, lokale Webseite zu steuern.
Die ältere Version benötigt alle eingaben manuell, durch erneutes hochladen des Codes.

### Diclaimer
